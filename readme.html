Bgro’s Anki Plugins¶

Anki is a great program for learning flashcards. It allows customization via Python-based plugins. This web page documents my plugins for Anki.

Quickstart¶

Install the add-ons you want to use in Anki via Tools > Add-Ons > Find & Install.
